## Les grands nombres de Topsxi
Ce repo contient le **document final** qui établit les **règles de construction** des grands nombres de Topsxi. Sauf qu'au cours du développement de ces règles, des branches vers d'autres opportunités mathématiques ont été découvertes. Comme celles-ci trouvent leur base dans la construction des grands nombres de Topsxi, alors leurs documents finaux sont aussi dans ce repo.

## Les prolongements

### Le théorème du trois parfait
Lors de la rédaction du document sur les grands nombres de Topsxi, une propriété a été remarquée sur certains grands nombres construits en exemple : ils étaient tous divisibles par 3 (sauf 1). Éveillant la curiosité, ce semblant de propriété a été assigné à une **conjecture** (nommée *conjecture du trois parfait*); sa démonstration formelle a été remise à plus tard. C'est ainsi que le document final énonce cette conjecture.\
Plus tard, cette conjecture a été formellement confirmée : maintenant, elle se nomme **théorème du trois parfait** et possède même un petit "corollaire" : le **cas de l'oublié**.\
La démonstration formelle, venue plus tard, fait l'objet d'un document différent de celui sur les grands nombres de Topsxi.
